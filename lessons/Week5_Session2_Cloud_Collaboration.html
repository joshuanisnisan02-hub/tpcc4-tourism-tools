Why Cloud

Why cloud over local files?

🌐 Access Anywhere

Phone, tablet, laptop, any location with internet

👥 Real-time Collab

Multiple people edit same doc simultaneously

💾 Auto-Save

Never lose work – every change saved instantly

📜 Version History

See all changes, revert if needed, track who edited what

🔗 Easy Sharing

Share link instead of emailing attachments back and forth

🔒 Backup

Files safe even if your device breaks, gets stolen, or crashes

Permissions

Access levels explained

LevelCan DoCan't DoTourism Use Case

CommenterView + add comments/suggestionsDirectly edit contentManager reviewing draft, giving feedback
EditorFull editing, add files, share with othersDelete the folder (only owner can)Team members working on project together
OwnerEverything + delete, transfer ownershipNothing restrictedProject lead, business owner

⚠️ Rule of thumb: Give the MINIMUM permission needed. Customer = Viewer. Supervisor = Commenter. Teammate = Editor. You = Owner.

Group Activity

Set up your team workspace

👥 Form groups of 4-5. Then complete these steps:

1. Choose a tour package destination (your group project for Weeks 5-7)
2. One person creates the shared folder on Google Drive
3. Create the 5 subfolders (01_Itinerary through 05_Presentation)
4. Share the folder with all group members (Editor) + instructor (Editor)
5. Each member creates ONE file inside the folder (any section)
6. Practice: Add a comment on someone else's file
7. Assign roles: Who handles Itinerary? Costing? Marketing? Communication? Presentation?

⏱️ 20 minutes
📧 Share with: joshua.nisnisan02@gmail.com (Editor)

Collaboration Tips

Working together smoothly

  • Use comments, not direct edits on others' work (unless agreed)
  • @mention teammates in comments to get their attention
  • Name files properly – if everyone has "Document1" it's chaos
  • Don't move others' files without asking
  • Resolve comments when addressed (don't leave hanging)
  • Check version history if something looks wrong – you can always revert
